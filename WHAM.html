NOAA Fisheries Integrated Toolbox

Tool: Woods Hole Assessment Model (WHAM)


Authors

Tim Miller and Brian Stock

Description

A state-space age-structured assessment model that can include state-space covariate models and effects on recruitment and internal estimation of biological reference points. See the vignettes provided with the package for example applications.
